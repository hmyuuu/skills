# Derivation Checker

Verify mathematical derivations step-by-step.

## User Request
$ARGUMENTS

## Workflow
1. Parse derivation into steps
2. Verify each step algebraically
3. Annotate errors/gaps
4. Human checks physical intuition

## Output
```markdown
### Step 1: [equation]
✓ Algebraically correct

### Step 2: [equation]
⚠ Warning: Sign error
```
