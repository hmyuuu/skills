# Academic Research

Research academic papers and questions, write typst documents with cetz visualizations, teach until human understands.

## User Request
$ARGUMENTS

## Workflow
1. Parse user's research question or paper URL
2. Web search for relevant papers and extract key insights
3. Ask human to judge source quality and relevance
4. Draft typst document with proper structure
5. Generate cetz plots for diagrams/graphs
6. Add citations, formatting, and flow
7. Check logic consistency and math validity
8. Loop until human confirms understanding

## Typst Template
```typst
#import "@preview/cetz:0.2.2"

= Title
== Background
== Key Concepts
== Analysis
== References
```

## Human Checkpoints
- Is the research question well-formed?
- Are sources trustworthy?
- Does the explanation make sense?
