# Experiment Logger

Log ML experiments with hyperparameters, metrics, and plots.

## User Request
$ARGUMENTS

## Workflow
1. Parse experiment name and parameters
2. Set up logging structure
3. Capture metrics and generate plots
4. Summarize results

## Output
```markdown
# Experiment: [name]
## Config
- learning_rate: 0.001
## Results
| Metric | Value |
|--------|-------|
| Loss   | 0.23  |
```
