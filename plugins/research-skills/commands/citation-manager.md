# Citation Manager

Find, format, and manage academic citations in BibTeX and typst formats.

## User Request
$ARGUMENTS

## Workflow
1. Parse paper title, DOI, or arxiv ID
2. Search Semantic Scholar, CrossRef, arxiv API
3. Format citation (bibtex/typst)
4. Validate and dedupe

## Output
```bibtex
@article{key,
  author = {},
  title = {},
  journal = {},
  year = {},
  doi = {}
}
```
